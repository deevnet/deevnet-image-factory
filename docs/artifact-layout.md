# Artifact Layout

Organization and naming conventions for build artifacts.

## Artifact Types

1. **Proxmox VM Templates** - Stored in Proxmox, named `<os>-<version>`
2. **Raspberry Pi Images** - `.img` files, named `<purpose>-<os>-<arch>.img`
3. **Bootable ISOs** - ISO files with embedded kickstart/autoinstall
4. **Builder Node Image** - Special image for the build infrastructure itself
